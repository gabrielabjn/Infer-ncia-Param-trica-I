Pelo teorema central do limite, sabemos que para qualquer populacao com media e variancia, a distribuicao das medias amostrais segue - para n suficientemente grande - uma normal com media equivalente a populacional e variancia igual a variancia populacional sobre o tamannho amostral n. Para a uniforme, percebe-se que os valores ficam centrados em torno da media logo em n = 20, revelando que o tamanho ‘suficientemente grande’ de n vai ser um valor mais baixo, provavelmente devido ao carater simetrico da distribuicao uniforme. A aproximacao da distribuicao pela normal parece ficar razoavel para n = 20, boa para n = 50 e muito boa para n >100. Quanto a variabilidade, essa reduz conforme n aumenta (e conforme era esperado pelo tcl).

As medias amostrais de Bernoulli(0.1) passam a apresentar relativa simetria apenas a partir de n = 50 (mas ainda nao podem ser aproximadas decentemente pela normal) e a ficarem centradas na media populacional apenas em n = 500. Isso revela a necessidade de se atribuir um valor maior a “n suficientemente grande” no caso dessa distribuicao (com esse parametro). Essa foi a distribuicao mais interessante de se trabalhar ate agora, pois foi possivel visualizar que para n = 20 a distribuicao nem se aproxima de uma normal ainda, o que pode causar erros grosseiros pelos mais desatentos. Variabilidade diminuindo conforme n cresce e conforme o esperado pelo tcl.
